# frontend-scaffold

A scaffold/generator to streamline setting up new frontend projects with config files, formatters and linters.

## Included tools

-   editorconfig
-   prettier
-   eslint
-   stylelint
-   husky
-   lint-staged

## Getting started

Run the following commands:

```
npx @loewen-digital/frontend-scaffold
npm install
```
